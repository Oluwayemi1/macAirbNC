Our endpoints are below to access our data
Just add the endpoint to our web address to get data related to the endpoint

Get "/api/properties"
This responds with an array of Properties;
Get "/api/properties/:id"
This responds with a single property with property ID specified in place of :id
Get "/api/properties/:id/reviews"
This responds with the reviews on the property with property ID specified in place of :id
Get "/api/users/:id"
This responds with details of the user with user ID specified in place of :id
Post "/api/properties/:id/reviews"
This creates a new review on property ID specified in place of :id and responds with the newly created review
Delete "/api/reviews/:id"
This deletes reviews done on property ID specified in place of :id
